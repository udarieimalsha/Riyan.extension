# Riyan pyRevit Extension

Professional Revit automation tools for link management and coordination.

## Installation

To install this extension, you must clone it into your pyRevit Extensions folder and ensure the folder name ends with `.extension`.

### Standard Installation
Run the following command in your terminal:
```powershell
git clone https://github.com/udarieimalsha/Riyan Riyan.extension
```

### Manual Installation
1. Download or clone this repository.
2. Rename the folder from `Riyan` to `Riyan.extension`.
3. Add the parent folder to pyRevit using the `pyRevit attach` command or via the pyRevit settings.
